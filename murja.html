feuerx.net - Murja - a blogging engine

A blogging engine that you can run from emacs

... even though the emacs-mode currently works, it is a monumental pain in the arse to install. While waiting for me to fix the emacs-mode, the blogging engine provides a web UI with ace-editor (used in, for example, Amazon's Cloud9) that tries to lie it's emacs.

The backend is written in Clojure, frontend is in Elm. It requires a PostgreSQL database for persistence.

The source, along its sort-of kind-of installation instructions are located in Github

ipad-client

There exists also an ipad client for murja, but I haven't bothered to package it to app store, as it is more like a learning experience about how to program Swift and SwiftUI than an actually useful client

Keywords: Clojure, Elm, Swift, SwiftUI
